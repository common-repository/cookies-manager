== Description ==

The Cookies Manager plugin allows you to set or get cookies used in a website. This plugin also allows users to set cookies as 'permanent' cookies. Its fully responsive plugin and
its based on twitter bootstrap framework. 

Demo Link : [Cookies Manager WordPress Plugin](http://cookiesmanager.fasterthemes.com)




== Installation ==

You can install Cookies Manager directly from the WordPress admin panel:

	1. Visit the Plugins > Add New and search for 'Cookies Manager'.
	2. Click to install.
	3. Once installed, activate and it is functional.
	
OR

Manual Installation:

	1. Download the plugin, then extract it.
	2. Activate the plugin through the 'Plugins' menu in WordPress
	
You're done! The plugin is ready to use, for more please check the plugin description.
